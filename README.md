# Final project

## Project Name
Final project

## Description
This project is an AI-based Emotion Detection web application developed using the IBM Watson NLP library and Flask. It analyzes customer feedback text and identifies the emotions expressed, such as joy, sadness, anger, fear, and disgust.

## Features
- Detects emotions from text input
- Identifies the dominant emotion
- Web interface built with Flask
- Error handling for invalid input
- Unit tested and statically analyzed

## Technologies Used
- Python
- Flask
- IBM Watson NLP
- HTML
- CSS
